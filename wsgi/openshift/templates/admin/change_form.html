Quick Start Guide

1) Select Obervation Type in the Form View

2) Use the Map Display in the Form View

3) Upload an Image in the Form View

4) Add Revision Notes in the Form View

5) Validate the Observation Entry in the Form View

6) View the History of the Observation Entry in the Form View






1) Select Observation Type in the Form View

  1. There are four types of observation:
    •      * Stationary - Observe in a single small location
    •      * Traveling - Observe while traveling along a trail
    •      * Casual - Observe in a random place
    •      * Area - Observe in a specified area
  2. You can choose the type of the observation in the observation type drop-down menu (see Figure 1.1).

    Figure 1.1: Observation type drop-down menu








2) Use the Map Display in the Form View

  1. The map display in the form view displays the location of the observation (see Figure 2.1).

    Figure 2.1: Observation map display
  2. The map marker  indicates your current location on the map. You can click and drag the map to change it's centered location.
  3. The map zoom control  lets you zoom in/out the map.
  4. The reset view button  lets you see the map at a global level.
  5. The map display is initially marked and centered at San Francisco, California, USA. You can change the current location by entering the Latitude and Longitude in the GPS fields (see Figure 2.2). The map display will automatically mark and center at the given GPS fields (see Figure 2.2).

    Figure 2.2: Observation GPS fields








3) Upload an Image in the Form View

  1. You can click on the file browser button  in the image field (see Figure 3.1) to upload an image.

    Figure 3.1: Observation image field
  2. A pop-up window of a file browser (see Figure 3.2) would appear for you to upload and manage your images.

    Figure 3.2: Observation image file browser
  3. You can sort the existing images by clicking on the column names. You can filter the existing images by using the filter menu options (see Figure 3.3) located at the upper right of the file browser (see Figure 3.2).
    After setting filters, in the "Results and Total" tab  located above and below the images, you can press the number total button to clear all filters.

    Figure 3.3: File browser filter menu options
    You can search for particular images by using the search box  located at the upper center-right of the file browser (see Figure 3.2).
  4. In the file browser (see Figure 3.2), you can press the "Upload" button  to start uploading an image file.
  5. You will be taken to the image upload page (see Figure 3.4) to upload the image file.

    Figure 3.4: File browser image upload page
  6. On the image upload page (see Figure 3.4), click on the "Help" tab to expand the help text pane (see Figure 3.5).

    Figure 3.5: File browser image upload page help text
  7. On the image upload page (see Figure 3.4), click the "Upload a File" button to upload an image file. You'll see the "Upload Completed" status (see Figure 3.6) if your image is successfully uploaded.

    Figure 3.6: File browser image upload page completed status
  8. From the image upload page (see Figure 3.4), click on the "FileBrowser" breadcrumb  located at the upper left corner to return to the file browser main page (see Figure 3.2).
  9. In the file browser (see Figure 3.2), you can click the "New Folder" button  to create a new folder for storing images.
  10. In the file browser (see Figure 3.2), you can click the change button  of an existing image to edit or delete the image.
  11. You will be taken to the image detail page (see Figure 3.7) to edit or delete the image.

    Figure 3.7: File browser image detail page
  12. On the image detail page (see Figure 3.7), use the "Edit Actions" drop-down menu (see Figure 3.8) to edit the image.

    Figure 3.8: File browser image detail page edit actions drop-down menu
  13. On the image detail page (see Figure 3.7), press the "Delete" button to delete the image.
  14. From the image detail page (see Figure 3.7), click on the "FileBrowser" breadcrumb  located at the upper left corner to return to the file browser main page (see Figure 3.2).
  15. In the file browser (see Figure 3.2), you can click the image resolution button  of an existing image to change the resolution of the image.
    A pop-up window with various resolution options (see Figure 3.9) would appear for you to select your preferred image resolution.

    Figure 3.9: File browser image resolution menu options
  16. In the file browser (see Figure 3.2), you can click the "Select" button  to attach the image to the observation entry. You will return to the form view after pressing the "Select" button .
  17. After you return to the form view, you can click on the image thumbnail icon (see Figure 3.10) to open the image in full-size in another browser tab/window.

    Figure 3.10: Observation image field thumbnail icon








4) Add Revision Notes in the Form View

  1. In the form view, you can add revision notes to an observation entry by clicking on the "Revision Notes" tab to expand the revision notes field (see Figure 4.1). The revision notes field is collapsed by default.

    Figure 4.1: Observation revision notes field
  2. Adding revision notes to an observation entry helps the validation process by providing manual annotations and data provenance information.








5) Validate the Observation Entry in the Form View

  1. After a thorough review of an observation entry, you can validate an observation entry for correct format by checking the "Validated" checkbox  in the validation field (see Figure 5.1).

    Figure 5.1: Observation validated field
  2. The validation process complements the uploading of observation entries by asserting their correctness of format.








6) View the History of the Observation Entry in the Form View

  1. You can check the history of an existing observation entry by clicking on the "History" button  located at the upper right corner of the form view.
  2. You will be taken to the observation entry history page (see Figure 6.1) to view the change history of the entry.

    Figure 6.1: Observation entry history page
  3. You can use breadcrumbs to return to previous pages.
