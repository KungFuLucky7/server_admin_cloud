1) Sort Observation Entries in the List View

  1. The observation list view is initially sorted by the species’ name alphabetically in ascending order .
  2. You can click on a column name to change the sorting order of the list by that column from ascending to descending , or reversely. You can sort the list by multiple columns.
  3. You can revert to the default order by clicking on an “x” button  on the column name.
  4. You can sort entries by time interval (date, month, year) by clicking on the time sorting options  at the upper left of the list view right above the column names.








2) Filter Observation Entries in the List View

  1. You can filter the list by clicking on the "Filter" button  at the upper right of the list view.
  2. A pop-up window with various filtering options (see Figure 2.1) would appear for you to select your preferred filters.

    Figure 2.1: Filter menu options
  3. After setting filters, in the "Results and Total" tab  located above and below the entries, you can press the number total button to clear all filters.








3) Search Observation Entries in the List View

  1. You can search for particular observation entries by entering a search query in the search box  to the right of the “Filter” button .




{% if request.user.is_superuser or perms.observations.change_observation and not perms.observations.readonly_observation %}




4) Delete Observation Entries in the List View

  1. You can select/deselect an observation entry by checking the checkbox  /  to the very left of the entry.
  2. You can select all entries by checking the checkbox  in the same row as the column names of the list view. 
  3. You can perform batch delete operation to the selected entries with the batch delete menu (see Figure 4.1) at the lower left of the list view. The batch delete menu displays the number of selected entries of the total.
    Figure 4.1: Batch delete menu
  4. You can click on the blank box of the batch delete menu and a pop-up window with the delete option (see Figure 4.2) would appear for you to apply the deletion of multiple entries.
    Figure 4.2: Batch delete menu popup








5) Import Observation Entries in the List View

  1. You can import observation entries by submitting a data file in an accepted format.
  2. You can start an import by clicking on the import button  at the upper right of the list view right above the search box .
  3. You will be taken to the import page (see Figure 5.1) to submit the data file.

    Figure 5.1: Import page
  4. Select the format of the data file in the "Format" drop-down menu (see Figure 5.2).

    Figure 5.2: Import format drop-down menu
  5. Press the "Submit" button on the import page (see Figure 5.1) to process the import.
  6. You will be presented with an import confirmation and a preview of imported entries. Press the "Confirm import" button  to complete the import.








6) Export Observation Entries in the List View

  1. You can export observation entries by saving filtered entries to a data file in a supported format.
  2. Apply filters in the list view to choose the particular observation entries for an export. Don't apply any filters for exporting all existing entries.
  3. You can start an export by clicking on the export button  at the upper right of the list view right above the search box .
  4. You will be taken to the export page (see Figure 6.1) to save the data file.

    Figure 6.1: Export page
  5. Select the format of the data file for the export in the "Format" drop-down menu (see Figure 6.2).

    Figure 6.2: Export format drop-down menu
  6. Press the "Submit" button on the export page (see Figure 6.1) to process the export.
  7. Your browser will automatically download the data file of the selected format to your "Downloads" directory.








7) Upload Observation Entries in the List View

  1. You can upload observation entries to an external archive server as a data file in a supported format. You can include image files attached to the entries as well.
  2. Apply filters in the list view to choose the particular observation entries to upload. Don't apply any filters for uploading all existing entries.
  3. You can start an upload of entries by clicking on the upload button  at the lower left of the list view below all obervation entries.
  4. You will be taken to the upload page (see Figure 7.1) to upload the entries.

    Figure 7.1: Upload page
  5. If you see the "No Internet" error message (see Figure 7.2) on the upload page, you don't have a valid Internet connection for uploading the entries.

    Figure 7.2: Upload page no Internet error message
  6. Select the format of the data file to upload in the "Format" drop-down menu (see Figure 7.3).

    Figure 7.3: Upload format drop-down menu
  7. You can include images attached to the selected entries by checking the checkbox  below the "Format" drop-down menu (see Figure 7.3).
  8. Press the "Upload" button on the upload page (see Figure 7.1) to process the uploading.
  9. If you see the failure message (see Figure 7.4), the uploading of the observation entries is not successful and you can try again later.

    Figure 7.4: Upload page failure message
  10. If you see the success message (see Figure 7.5), the uploading of the observation entries is successful!

    Figure 7.5: Upload page success message
